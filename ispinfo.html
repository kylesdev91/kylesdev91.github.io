Conclusions: What went as expected; what did not and why?

Expected: Everybody in the group had the access to the files in Github after creating in KQuiz directory.

Problem 1: The KQuiz title was hidden under the background, with no way to show up. Solution 1: We had to render fill style and stroke style for the text to show up on the background

Problem 2: Another problem we faced is how to randomize the questions/numbers in the IGIS browser. Solution 2: Inside of BackgroundLayer.swift, we created a variable that stores a random number between 1 and 49. Then, we assigned the questionIndex to the variable randomQuestion instead of a fixed number.

Problem 3: We did not know how to code buttons in Swift.Solution 3: We imported Scenes Controls from Mr. Ben github codermerlin and made four buttons, one for each of the answer choices.

Problem 4: Words kept getting on top of each other when the next button is clicked. Solution 4: We decided to create ten clear buttons manually One for each question
